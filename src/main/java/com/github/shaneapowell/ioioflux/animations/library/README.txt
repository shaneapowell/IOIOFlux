Any custom animations or animation factories go here.

Please don't add to the main AnimationFactory class.  Instead, create and add your own factory class.  Name it after yourself.
for Example, AnimationFactoryJohn.java, or AnimationFactoryMarty

Any concrete complex animation sequences you want to create outside of a factory, place those here also.
For example, I created the "Cylon" animation class and placed it in here.
I also added a method to the main factory just for simplicity.
